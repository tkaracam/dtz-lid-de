# IONOS'ta Yönlendirmeyi Kaldır

## 🗑️ Yönlendirmeyi Sil

### Adım 1: Domain Yönetimine Git
```
IONOS Panel: https://my.ionos.com/domain-center
→ dtz-lid.de bul
→ "Verwalten" (Yönet)
```

### Adım 2: Yönlendirme Menüsü
Sol menüde şunu ara:
```
"Weiterleitung" veya "Redirect" veya "Forwarding"
```

### Adım 3: Mevcut Yönlendirmeyi Sil
Eğer şöyle bir şey varsa:
```
Von: dtz-lid.de
Zu:  https://dtz-lid.de/member.html
```

→ **"Löschen"** veya **"Deaktivieren"** tıkla

### Adım 4: Onayla
```
"Sind Sie sicher?" → "Ja, löschen"
```

---

## 🔄 Alternatif: Yönlendirmeyi Değiştir

Sil butonu yoksa, yönlendirmeyi değiştir:

```
Von: dtz-lid.de
Zu:  https://www.dtz-lid.de          ← member.html YOK!

Tip: 301 (Permanent)
```

---

## ✅ Kontrol Et

Yönlendirme silindi mi?
```
IONOS → dtz-lid.de → Weiterleitung
→ "Keine Weiterleitung vorhanden" görmelisin
```

---

## 🆘 Yönlendirme Menüsü Yok mu?

Paket bazlı yönlendirme olabilir:
```
IONOS → Meine Produkte
→ "MyWebsite" veya "Homepage-Baukasten" veya "Redirect"
→ Kündigen / Löschen
```
